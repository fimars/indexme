## Index ME!
Like windows-trees, but more format.

### Installtion
```bash
npm install indexme -g
```
### Useage
```bash
âž” indexme --help

  Usage: indexme [options] path


  Options:

```

### Recipes

Index your markdown notes.
```bash
index notes -i README.md > notes/README.md
```
